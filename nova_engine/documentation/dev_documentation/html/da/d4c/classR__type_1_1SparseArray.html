Detailed Description

template<typename Component>
class R_type::SparseArray< Component >

A Sparse array class to store sparse components efficiently.

This class is designed to store components that are assigned to a large set of entities in an Entity-Component-System (ECS) architecture. It uses one vectors to store components for every entities that have them, optimizing time usage for sparse components.

Template Parameters
TThe type of component to be stored in the Sparse array.
