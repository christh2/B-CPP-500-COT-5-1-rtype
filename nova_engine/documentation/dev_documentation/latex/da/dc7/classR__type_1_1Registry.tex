\doxysubsection{Detailed Description}
A class that manages entities and components in a hybrid array system. 

The \mbox{\hyperlink{classR__type_1_1Registry}{Registry}} provides methods for creating and destroying entities, registering and managing components, and executing systems. It supports various systems like standard, event-\/based, and rendering systems, which can operate on entities and their components.
